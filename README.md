# mc-cdm

## Развёртывание проекта

### Настройка конфигов

Конфиги для настройки лежат в директории [/config/](/config/)

1. Импортируем [дамп БД](/mc-cdm.sql) MySQL.

2. В [конфиге подключения к БД](/config/db_conn.php) необходимо указать реквизиты для подключения к базе данных.

```php
const DB_DRIVER = "mysql"; # Не менять
const DB_HOST = "MySQL-8.2";
const DB_NAME = "library";
const DB_USER = "root";
const DB_PASSWORD = "";
```

3. К [config.php](/config/config.php) необходимо указать имя web приложения и базовое имся страницы. Остальные параметры, как правило не нуждается в изменении.

```php
const APP_DOMEN = "mc-cdm";
const BASE_PAGE_NAME = "МЦ | ";
```

> [!NOTE]
> "BASE_PATH" предусмотрен для случаев, когда корневая директория сайта находится глубже, чем указано в конфигурациях сервера apache. (Таким образом контрим приколы хоста, который нам предоставляет АГУ)
